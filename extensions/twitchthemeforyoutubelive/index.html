Twitch theme for YouTube live

Some settings for YouTube live chat from Twitch.

"Twitch theme for YouTube live" enhances YouTube with new features, and more.
"Twitch theme for YouTube live" aims to add new features to YouTube website.

Here's just some of the features we add to chat:

• Hide user avatar in chat
• Colored usernames in the chat
• Hide visual noise like engagement message
• Highlight message on hover

How to use:
• Just install and visit YouTube to begin using the add-on.
• Chat settings can be found by clicking on the three dots in the upper right corner of the chat.
• Options can be configured in the chat settings.
• We advise you to refresh the page after opening a stream!

If you find a bug, please contact the developer!

Twitch VOD Player

Allow the browser to play HLS video urls (m3u8).

Built-in M3U8 player, play HLS in your browser and watch Twitch VOD

How to use M3U8:
• Paste the .m3u8 or video transfer link into the navigation bar.
• Then the tab with the player will open

How to use to viewing Twitch VOD:
• Just install and visit Twitch videos to start using the add-on.
• Right click on VOD and select Twitch VOD Player.
• Then the tab with the player will open
